# Lesson entry template

Use this format when appending to `memory/LESSONS.md`:

```
## L-<next-number> — <short-name>
Date: <YYYY-MM-DD>
Task: <task_id>
What went wrong: <one or two sentences>
Root cause: <real reason, not symptom>
Avoid by: <concrete rule for next time>
Seen N times: 1
```

Rules:
- Increment `Seen N times` if this lesson repeats, do not duplicate the entry
- At 3+ occurrences, Auditor promotes this to a GOTCHA during weekly review
- Delete if the underlying code has changed and lesson no longer applies
