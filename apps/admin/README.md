# @dripwell/admin

DripWell Admin Panel — desktop interface for super users.

## Purpose

Clinic owners and lead providers use this interface to manage their catalog, providers, settings, and view analytics.

## Development

```bash
# From repo root
pnpm dev:admin    # Starts the admin app on http://localhost:3001
```

## Structure

- `src/main.ts` — Admin app entry point

## Notes

- Built with Vite + TypeScript
- Imports shared types from `@dripwell/shared`
